# 💊 Aprajita Kumari — Physiotherapist Portfolio

A modern, single-page portfolio website for **Aprajita Kumari**, a dedicated Physiotherapist (BPT) based in New Delhi, India. The site showcases her clinical experience, education, skills, specialized training, and community involvement — all wrapped in a premium, animated design.

---

## ✨ Features

| Feature | Description |
|---|---|
| **Animated Preloader** | Heartbeat-line SVG animation while the page loads |
| **Hero Section** | Typing-text effect, floating medical icons, particle background, and animated profile photo frame |
| **Smooth Scrolling** | One-page navigation with scroll-spy active-link highlighting |
| **Responsive Navbar** | Glassmorphism nav bar with hamburger toggle for mobile |
| **Timeline Layout** | Vertical timeline for education history with alternating left/right reveals |
| **Skills Dashboard** | Animated progress bars with scroll-triggered fill animations |
| **Department Cards** | Grid of 6 clinical departments with hover effects |
| **Workshops & Camps** | Card-based layouts for specialized training and community healthcare camps |
| **Conference Section** | National & international conference attendance highlights |
| **Contact Form** | Floating-label form with validation and success feedback |
| **Back-to-Top Button** | Appears on scroll with smooth return to top |
| **Scroll Animations** | Intersection Observer–driven reveal effects (fade-up, slide-left, slide-right) |
| **Fully Responsive** | Mobile-first design that adapts from phones to large desktops |

---

## 🛠️ Tech Stack

- **HTML5** — Semantic markup with SEO meta tags
- **CSS3** — Vanilla CSS with custom properties, glassmorphism, gradients, keyframe animations, and media queries

## 🚀 Getting Started

### Prerequisites

No build tools or dependencies required — this is a static HTML/CSS/JS site.

### Run Locally

1. **Clone or download** the repository:
   ```bash
   git clone https://github.com/<your-username>/aprajita-portfolio.git
   cd aprajita-portfolio
   ```

2. **Open in browser** — simply open `index.html` in any modern browser:
   ```bash
   # macOS
   open index.html

   # Windows
   start index.html

   # Linux
   xdg-open index.html
   ```

   Or use a local dev server for a better experience:
   ```bash
   # Using Python
   python -m http.server 8000

   # Using Node.js (npx)
   npx serve .
   ```

3. **Visit** `http://localhost:8000` in your browser.
